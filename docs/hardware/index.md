# Gambos Documentation

Technical documentation for the Gambos embedded project.

Gambos combines:

- custom hardware design (KiCad schematics + PCB)
- STM32 firmware development (CMake + FreeRTOS)

## Project status

- Hardware: PCB manufactured, assembly/bring-up pending
- Software: build environment and firmware project in progress
